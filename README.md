# RAG Architectures Lab

A research-style project implementing multiple **Retrieval-Augmented Generation (RAG) architectures** using Python, LangChain, ChromaDB and Ollama.

This repository explores different RAG pipelines used in modern AI systems and compares how each architecture retrieves and generates knowledge.

---

# Implemented RAG Architectures

This project includes multiple advanced RAG implementations:

### 1. Standard RAG
Basic retrieval augmented generation pipeline.

### 2. Advanced RAG
Improved retrieval pipeline with better context construction.

### 3. Corrective RAG (CRAG)
Detects retrieval failures and corrects them using external knowledge.

### 4. Self-RAG
The model evaluates its own answers and improves responses.

### 5. Fusion RAG
Combines multiple retrieval results to produce a better final answer.

### 6. Speculative RAG
Uses speculative generation techniques to improve response speed.

### 7. Web RAG
Fallback web search when the vector database lacks information.

### 8. Knowledge Graph RAG
Visualizes relationships between concepts extracted from documents.
